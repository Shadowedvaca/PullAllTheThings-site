Discord Bot

Connect a Discord bot for role sync, DMs, and announcements.

  1. Open the Discord Developer Portal
    discord.com/developers/applications →
  2. Create a New Application → click "Bot" in the sidebar → click "Add Bot"
  3. Under Privileged Gateway Intents, enable Server Members Intent
  4. Click Reset Token → copy the token and paste it below

Use the invite link below to add the bot to your Discord server. Once added, select your server below.
