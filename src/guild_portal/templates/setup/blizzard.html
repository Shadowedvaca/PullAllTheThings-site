Blizzard API

Connect to the Blizzard API to sync your guild roster.

  1. Open the Blizzard Developer Portal
    develop.battle.net/access/clients →
  2. Click Create Client → give it a name (e.g., "Guild Portal")
  3. Copy your Client ID and Client Secret

Use the API slug format: lowercase, no apostrophes (e.g., "senjin" for Sen'jin, "illidan" for Illidan)

Use lowercase with hyphens (e.g., "pull-all-the-things")
