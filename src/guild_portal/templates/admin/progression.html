M+ Season

The Blizzard M+ season ID is set on the active raid season.

Go to Admin → Reference Tables and enter the Blizzard integer season ID in the M+ Season ID column of the active raid season row.

Blizzard uses plain integers (13, 14, 15…), not dot notation. Find the current season ID at the Blizzard API season index (requires a valid access token, or check Wowhead).

Progression syncs every 6 hours. Achievement sync + snapshots run weekly (Sunday 4:30 AM UTC).
