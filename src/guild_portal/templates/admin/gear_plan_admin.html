Raw Data Refresh

Populate the landing tables from the Blizzard API and Wowhead. Flush & Fill truncates all landing tables before refilling — use at season start. Catch Up skips the truncate and only pulls new/missing items.

{# ── Section B — BIS Scraping ─────────────────────────────────────────────── #}
B

BIS Scraping

Scrape u.gg and Wowhead BIS recommendations for all specs. Discover URLs finds scrape targets for the current season. Test Fetch runs a single Blood DK scrape for quick verification. Sync All scrapes every active target. Gap Fill scrapes only targets with no raw data or last fetched >7 days ago. Re-Sync Errors retries only failed targets.

C

Cleanup and Populate

Enrich & Classify fetches Wowhead tooltips for any unenriched items, runs tier token processing, then rebuilds the full enrichment schema (items → sources → BIS entries → trinket ratings → category classification). Safe to re-run at any time. Populate All Plans creates gear plans for all in-guild characters that don't have one and fills unlocked slots from Wowhead Overall.
