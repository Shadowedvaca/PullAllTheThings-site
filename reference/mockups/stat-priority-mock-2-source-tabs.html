Mock 2 — Source Tabs

Concept: A tab row lets the user pick which guide's stat priority they want to see. All sources are visible; you pick the one you trust. When Icy Veins is active, hero talent sub-tabs appear since IV splits by hero talent path. The Archon tab adds weight bars and a summary target line. The strip is sticky; the full tab row + stat strip scroll-pins together.

Pros: Each source is always visible and selectable. Easy to compare by tab-switching. Archon's quantitative data is surfaced naturally on its own tab. Hero talent sub-tabs only appear when relevant. Player controls exactly which guide opinion they're seeing.
Cons: Requires the user to know which source to trust. No "summary" view — they must tab through to compare. Sources with no data for a given content type need graceful empty states. Two rows of tabs (source + hero talent) adds height to the sticky area.
