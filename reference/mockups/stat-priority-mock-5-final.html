Tooltips: Hovering any block shows which sources rank it at which position, plus any operator notes. Green consensus blocks still get a confirming tooltip for the curious player.

Color band legend: One horizontal bar, five segments from green (5/5) through yellow to red (1/5). Faster to read than individual swatches — your eye maps the block border color directly to its band segment.

Operator dot: Amber dot below a consensus operator means at least one source uses a different symbol there. The adjacent block tooltip names the offending source. The detail drill shows each source's actual operator.
