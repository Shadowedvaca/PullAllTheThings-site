Pros: Rich at-a-glance signal about consensus quality without adding a tab row or expand panel. When all guides agree, the strip looks clean and fully gold. Disagreements are visible but not alarming. The pip row is compact — doesn't add much height.
Cons: Color meaning (green/yellow/red) requires a legend. The tooltip detail (which source says what) requires interaction. Pip row adds a second line under each block — the strip is slightly taller than options 1 or 2. "3/5 rank #2" requires understanding the scale.
